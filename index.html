BST Height

一棵二叉树的高度是其根结点(root) 至其 最深的结点(the deepest node) 的路径长度.

△一棵只有根结点的树，其高度为0

BST Height Algorithm
BST高度计算算法 (pesudo code)

Input: Binary Tree T
Output: Height of T

if T = ∅   then
   return -1
else
   return max(BSTHeight($T_L$), BSTHeight($T_R$)) + 1
end if

Pre-order

在当前子树的左右子树被遍历之前，先visit其根结点(root)。
既每个结点按照以下优先级进行操作。
visit -> left -> right

output: 16 -> 6 -> 4 -> 1 -> 5 -> 12 -> 26 -> 20 -> 28 -> 27

In-order

left -> visit -> right

output: 1 -> 4 -> 5 -> 6 -> 12 -> 16 -> 20 -> 26 -> 27 -> 28

Post-order

left -> right -> visit

output: 1 -> 5 -> 4 -> 12 -> 6 -> 20 -> 27 -> 28 -> 26 -> 16
